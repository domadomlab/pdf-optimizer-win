# PDF Optimizer Suite - Отчет о разработке (19 января 2026)

## Статус проекта: v4.2.1 (Win) / v3.2.0 (Linux) - "Scientific Parity Update"

### Ключевые достижения (v4.2.1 -> v4.2.1 / v3.2.0):

1.  **Интеллектуальный стандарт 150 DPI:**
    *   Технология **Trellis-Quantization Mimic** (Lanczos + Resize 95% + Unsharp) перенесена из экспериментального режима "Extreme" в основной рабочий режим **Email (150 DPI)**.
    *   Это позволило значительно улучшить качество сжатия офисных документов: фон стал чище, буквы контрастнее, а вес файлов уменьшился еще на ~15% при сохранении высокого качества (Quality 70).

2.  **Исправление метаданных пакета (Linux):**
    *   Исправлено несоответствие версии в поле `Description` файла `control` для `.deb` пакета.
    *   Теперь описание четко указывает на версию **v3.2.0** и содержит актуальный список инноваций.

3.  **Полная синхронизация логики:**
    *   Обе платформы используют идентичные цепочки команд ImageMagick для режимов Extreme (Quality 40) и Email (Quality 70).

### Технические нюансы (Memory Update):

*   **150 DPI Logic:** `convert -density 150 -filter Lanczos -distort Resize 95% -unsharp 0x0.5 -sampling-factor 4:2:0 -quality 70`.
*   **Package Integrity:** Всегда проверять текстовые описания в `control` (Debian) и `installer.nsi` (Windows) при смене версий. Описание должно быть зеркалом технических полей.
*   **Version Consistency:** Актуальные версии на текущий момент: **Windows v4.2.1**, **Linux v3.2.0**.
