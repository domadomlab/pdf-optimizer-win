# PDF Optimizer Suite v3.0.0

Профессиональный программный комплекс для оптимизации PDF-файлов через контекстное меню Windows 10/11.

## Основные возможности:
- **Интеграция в проводник**: Правый клик по любому PDF и выбор нужного DPI (150, 200, 300).
- **All-in-One**: Автоматическая установка всех зависимостей (Python, ImageMagick, Ghostscript).
- **Умный поиск**: Скрипты автоматически находят интерпретатор и движки обработки.
- **Глубокая очистка**: Полное удаление всех компонентов и записей реестра при деинсталляции.

## Компоненты:
- **Python 3.12**: Логика и управление.
- **ImageMagick**: Движок обработки изображений.
- **Ghostscript**: Рендеринг PDF (Delegate).

## Установка:
Запустите `PDF_Optimizer_Suite_v3.0.0_Installer.exe` от имени администратора. Весь процесс полностью автоматизирован.
