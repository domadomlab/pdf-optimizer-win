# PDF Optimizer Suite v3.4.4

Профессиональный программный комплекс для оптимизации PDF-файлов через контекстное меню Windows 10/11.

## Основные возможности:
- **Интеграция в проводник**: Компактное и понятное меню по правому клику:
    - **PDF: Eco (75 dpi)** — Максимальное сжатие для архивов.
    - **PDF: Email (150 dpi)** — Баланс для отправки по почте.
    - **PDF: Print (200 dpi)** — Оптимально для офисной печати.
    - **PDF: High (300 dpi)** — Высокое качество.
- **Надежность (Triple Registry Injection)**: Меню регистрируется в трех ветках реестра одновременно (HKLM, HKCU System, HKCU Direct), что гарантирует его появление даже при жестких политиках безопасности.
- **All-in-One**: Автоматическая установка и настройка Python 3.12, ImageMagick и Ghostscript.
- **Self-Healing**: Инсталлятор умеет "чинить" сломанные установки Python и восстанавливать ассоциации.
- **Глубокая очистка**: Полное удаление всех компонентов, временных файлов и записей реестра при деинсталляции.

## Компоненты:
- **Python 3.12**: Логика и управление.
- **ImageMagick**: Движок обработки изображений.
- **Ghostscript**: Рендеринг PDF (Delegate).

## Установка:
Запустите `PDF_Optimizer_Suite_v3.4.4_Installer.exe` от имени администратора. 
Весь процесс полностью автоматизирован и занимает менее минуты.
